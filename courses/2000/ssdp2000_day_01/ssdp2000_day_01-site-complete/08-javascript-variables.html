JavaScript Variables

About JavaScript Variables

JavaScript variables can be set with with either the "var", "let" or "const" variables.

When to use var, let and const

  • var: Use var if you anticipate that your script will be running in pre-2015 browsers. This is mainly IE10,9 and 8.
  • let: Use let if the value of the variable will change while the script is running.
  • const: Use const if the value of the variable will not change while the script is running.

My Personal Method

I default to always using const first, and only use let if the variable value might change during the running of the script.
